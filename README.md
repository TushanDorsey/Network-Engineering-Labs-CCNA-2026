
# Network-Engineering-Labs-CCNA-2026

## Overview

Welcome to my Network Engineering Lab portfolio.

This repository documents my hands-on networking journey as I prepare for the Cisco Certified Network Associate (CCNA) certification and continue developing the practical skills required for Network Engineering roles.

The goal of this repository is to move beyond theory by building, documenting, and troubleshooting real-world networking scenarios using Cisco Packet Tracer and other networking tools.

Throughout this journey, I will be working through Jeremy's IT Lab CCNA course while expanding on the concepts through additional documentation, network diagrams, configuration examples, and lessons learned.

---

## Objectives

* Develop practical networking skills through hands-on labs
* Prepare for the Cisco CCNA (200-301) certification
* Build a professional networking portfolio
* Strengthen enterprise networking knowledge
* Document network troubleshooting and design concepts
* Share progress publicly while learning
