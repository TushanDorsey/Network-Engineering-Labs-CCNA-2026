# 🧪 CCNA Labs

## Overview

This directory contains all hands-on networking labs completed throughout my CCNA studies.

Each lab focuses on a specific networking concept and includes documentation, screenshots, network topologies, Packet Tracer files, and key takeaways from the exercise.

The goal of these labs is to reinforce networking concepts through practical implementation rather than theory alone.

## Learning Approach

For each lab I aim to:

1. Understand the concept.
2. Build the topology.
3. Configure the devices.
4. Verify connectivity.
5. Document the process.
6. Record key lessons learned.

This approach helps reinforce technical knowledge while building a portfolio of practical networking experience.

---

## Objective

The ultimate goal of these labs is to:

* Pass the Cisco CCNA (200-301)
* Strengthen networking fundamentals
* Build real-world troubleshooting skills
* Develop a professional Network Engineering portfolio
